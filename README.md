# 🚛 TruckerService - Sistema de Manutenção de Caminhões

Este projeto consiste em uma **Web API** desenvolvida para a disciplina de **Desenvolvimento Web Avançado**, sob orientação do **Prof. Marlon**.

O sistema tem como objetivo gerenciar a manutenção de frotas de caminhões, permitindo o controle de **veículos**, **mecânicos** e **ordens de serviço (OS)**.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** C# (.NET SDK)  
- **Framework:** ASP.NET Web API  
- **ORM:** Entity Framework Core  
- **Banco de Dados:** SQLite  
- **Arquitetura:** Repository Pattern  


---

## 📋 Entidades Principais

O sistema utiliza as seguintes entidades relacionadas:

### 🚚 Caminhao
Cadastro dos veículos da frota.

### 👨‍🔧 Mecanico
Cadastro dos profissionais responsáveis pela manutenção.

### 📄 OrdemServico
Gestão da manutenção, vinculando um veículo e um mecânico  
(opcional no momento da abertura).
